# Admin UX Patterns

WordPress admin interfaces work best when they feel like WordPress, not a disconnected custom app.

## Review For

- Clear page titles and section labels
- Stable notice behavior
- Predictable save flow
- Screen-specific actions
- Lightweight, scoped asset loading

## Prefer

- Core admin styles and layout patterns
- Simple tables and forms over overbuilt JS interfaces
- Progressive enhancement where JavaScript adds convenience, not basic usability
